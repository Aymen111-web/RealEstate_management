 Real Estate Management System

A web-based Real Estate Management system built with **HTML, Bootstrap, and JavaScript** for the frontend, and **PHP** for the backend. This project allows users to manage property listings efficiently and provides a simple interface for property management.

---

Features

- Add, view, edit, and delete property listings
- Responsive design using Bootstrap
- Dynamic frontend interactions using JavaScript
- PHP backend for server-side operations
- Organized project structure for easy maintenance

---

 Technologies Used

- Frontend: HTML, CSS, Bootstrap, JavaScript  
- Backend: PHP  
- Database: ( MySQL)
